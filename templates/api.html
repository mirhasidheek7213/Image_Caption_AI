API Documentation

Integrate our image captioning capability into your applications using our API.

Endpoint: /caption

Method: POST

Content-Type: multipart/form-data

Request Parameters

Parameter Type Description
image File Image file to caption (JPG, PNG, GIF)

Response

{
  "caption": "A person hiking on a mountain trail with a scenic valley view"
}

Example Usage

// JavaScript fetch API example
fetch('/caption', {
  method: 'POST',
  body: formData // formData containing the image file
})
.then(response => response.json())
.then(data => console.log(data.caption))
