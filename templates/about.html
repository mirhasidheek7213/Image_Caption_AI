About ImageCaption AI

ImageCaption AI is a image captioning system that uses neural networks to generate descriptive captions for images.

How it Works

Our system uses a combination of combination of ResNet18 and Transformer decoder:

  • A ResNet18 convolutional neural network extracts visual features from the uploaded image
  • A Transformer decoder generates natural language descriptions based on these features
  • Beam search is used to produce high-quality captions that accurately describe the image content

Technology Stack

Our application is built using:

  • PyTorch for deep learning
  • Flask for the web framework
  • Modern HTML/CSS/JavaScript for the frontend
